# Hand Gesture Recognition using CNN

## Overview

This project implements a Convolutional Neural Network (CNN) to recognize and classify different hand gestures from image data. The objective is to enable intuitive human-computer interaction through gesture-based control systems.

## Dataset

Dataset: LeapGestRecog Dataset

Source: https://www.kaggle.com/datasets/gti-upm/leapgestrecog

## Technologies Used

* Python
* TensorFlow / Keras
* OpenCV
* NumPy
* Matplotlib
* Scikit-learn
* Streamlit

## Project Workflow

* Dataset Loading
* Image Preprocessing
* Data Normalization
* CNN Model Building
* Model Training
* Model Evaluation
* Gesture Prediction
* Streamlit Application Development

## Features

* Hand gesture image classification
* CNN-based deep learning model
* Image upload and prediction
* Confidence score display
* Prediction probability visualization
* Interactive Streamlit interface

## Results

The model successfully recognizes and classifies multiple hand gestures from image data. It predicts gesture labels along with confidence scores, enabling gesture-based interaction and control systems.

## How to Run

Clone the repository

Install dependencies

pip install -r requirements.txt

Run the application

streamlit run app.py
